For both HDRP and URP templates you need to use Unity version 2019.4.0 or newer. 

For more information about HDRP and URP, please read the online documentation: https://docs.google.com/document/d/1JuJERUs9bHl0VlrwWS2Npzt2NnEEhCTISLQhiKrBZ54
If you have any questions and concerns do not hesitate to contact us at: contact@angrymesh.com

--------------------------------------------------------------------------------

HDRP - HOW TO USE:
1. Create a new empty project using HDRP Template.
2. Go to Windows > Package Manager and select High Definition RP.
	- For Unity 2021.2.0 or for a newer version of unity, update the High Definition RP to version 12.1.0 or above.
	- For Unity 2020.2.0, update the High Definition RP to version 10.2.2 or above.
	- For Unity 2019.4.0, update the High Definition RP to version 7.2.1 or above.

3. Import Procedural Sky (Windows > Package Manager and select High Definition RP)
4. Import the Winter Environment package in the project.
5. Import the HDRP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP. 

Note: 
For Unity 2021.2.0 or newer, HDRP version 12.1.0 or above - You need to use HDRP template 12.1.0.unitypackage
For Unity 2020.2.0, HDRP version 10.2.2 or above - You need to use HDRP template 10.2.2.unitypackage
For Unity 2019.4.0, HDRP version 7.2.1 or above - You need to use HDRP template 7.2.1.unitypackage

--------------------------------------------------------------------------------

URP - HOW TO USE:
1. Create a new empty project using URP Template.
2. Go to Windows > Package Manager and select Universal RP.
	- For Unity 2021.2.0 or for a newer version of unity, update the Universal RP to version 12.1.0 or above.
	- For Unity 2020.2.0, update the Universal RP to version 10.2.2 or above.
	- For Unity 2019.4.0, update the Universal RP to version 7.2.1 or above.

3. Import the Winter Environment package in the project. 
4. Import the URP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP. 

Note: 
For Unity 2020.2.x or newer, URP version 10.2.2 or above - You need to use URP template 10.2.2.unitypackage
For Unity 2019.3.6.x, URP version 7.2.1 or above - You need to use URP template 7.2.1.unitypackage
For Unity 2019.3.x, URP version 7.1.5 or above - You need to use URP template 7.1.5 (Legacy).unitypackage
